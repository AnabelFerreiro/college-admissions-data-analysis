Data Notes

Additional context about the dataset, methodology, and AI workflow used in the Retail Performance Analysis: Sales, Profitability, and Discount Impact project.

Project Context

This project was completed as the CAI 3801 final project. It combines an interactive Tableau dashboard with an AI agent workflow that turns dashboard findings into an automated executive summary email.

Business Question

How do discounting practices affect sales and profitability across categories and regions, and how can analytics-driven recommendations be communicated to decision-makers efficiently?

Dataset

  • Transactional retail data including sales, profit, discount, category, sub-category, and region
  • Order-level dates used to support time-series forecasting
  • Discount rates analyzed alongside profit margin to assess pricing impact

Methodology

  • Built a 5-chart Tableau dashboard with cross-chart filter actions for drilling from region to sub-category
  • Added a What-If parameter to simulate the profit impact of adjusting discount rates
  • Used Tableau's forecasting feature to project near-term sales trends
  • Exported dashboard findings to a PDF report for downstream automation

AI Workflow

  • Built a 2-agent sequential workflow in CrewAI Studio
  • Agent 1 used PDFSearchTool to extract verified findings from the exported dashboard report
  • Agent 2 (GPT-4.1) drafted a professional executive summary and sent it via the Gmail integration
  • The pipeline runs end-to-end with no manual write-up required

Key Insight

Profitability and discounting are closely linked: several sub-categories become unprofitable past a certain discount threshold. The What-If scenario showed that trimming discounts on low-margin sub-categories could recover meaningful profit without significant sales loss.

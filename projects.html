College Admissions & Performance Analysis

This project explores admission outcomes, compares SAT vs High School GPA as predictors of first-year college performance, and evaluates regression strength using R² models.

Business Question

Which metric is a stronger predictor of first-year college GPA—SAT scores or High School GPA— and how can admissions teams use that insight to improve decision-making?

Approach

  • Explored relationships between admissions variables and first-year GPA.
  • Compared regression strength using R² values across predictors.
  • Built an interactive dashboard to communicate trends and outcomes.

Key Findings

Stronger predictor

SAT effect
SAT scores had moderate correlation but lower explanatory strength.
Decision support
Regression analysis clarified which metric better predicts academic success.

# college-admissions-data-analysis
Exploratory analysis of academic factors influencing college admissions and first-year performance

Analysis Questions
    What is the overall admission rate?
    How do SAT Total scores differ between admitted and non-admitted students?
    How does high school GPA compare between admitted and non-admitted students?
    Is there an approximate SAT or GPA threshold associated with admission?
    How does high school GPA relate to first-year GPA?
    Which pre-admission metric (SAT or HS GPA) better predicts first-year performance?
    Are there outliers where students with strong academic profiles were not admitted?
    What do these patterns suggest about the consistency of the admissions process?

Analysis Plan

    Clean and validate the dataset
    Perform descriptive analysis using pivot tables
    Compare academic metrics by admission outcome
    Visualize relationships between pre-admission metrics and first-year GPA
    Summarize insights and recommendations

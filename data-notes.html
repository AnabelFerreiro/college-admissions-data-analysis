Data Notes

Additional context about the dataset, methodology, and assumptions used in the College Admissions & Performance Analysis project.

Project Context

This project was completed as part of an in-class analytics exercise. The objective was to evaluate which metric ‚Äî SAT score or High School GPA ‚Äî better predicts first-year college GPA using regression analysis.

Business Question

Which metric is a stronger predictor of first-year academic performance, and how could admissions teams use that insight to improve decision-making?

Dataset

  • Dataset included student-level SAT scores, High School GPA, and first-year college GPA.
  • Variables were standardized for comparison.
  • Missing or incomplete observations were excluded where necessary.

Methodology

  • Built regression models comparing SAT and HS GPA as predictors.
  • Evaluated explanatory strength using R¬≤ values.
  • Compared predictive power across variables to determine stronger indicator.
